# Heroes of Silent Earth: 3

A fan-made browser game set in the universe of **The Amory Wars** by Coheed and Cambria.

Play as Claudio Kilgannon, Chase, and the companions of Heaven's Fence across multiple levels — from the streets of Newo to the Godder Dam and beyond.

**[Play it here →](https://yourusername.github.io/heroes-of-silent-earth-3)**  
*(update this link after GitHub Pages is live)*

---

## How to Play

**Desktop:** Arrow keys to move, F to attack, G for power blast  
**Mobile:** Drag left side to move, tap right side to attack, double-tap right side for power blast

---

## Running Locally

No build tools needed. Just serve the files over HTTP:

```bash
# Python
python3 -m http.server 8000

# Node
npx serve .

# VS Code
Install Live Server extension → right-click index.html → Open with Live Server
```

Then open `http://localhost:8000` in your browser.
